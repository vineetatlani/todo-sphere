My Social Media App

My Social Media App is a platform for users to manage their to-do lists and create posts with comments. Our APIs are easy to use and allow developers to build custom applications that integrate with our platform.

  • User Sign Up: /user
  • User Log In: /login
  • Create Todo: /todo
  • Get Todo: /todo/:id
  • Edit Todo: /todo/:id
  • Delete Todo: /todo/:id
  • Create Post: /post
  • Get Post: /post/:id
  • Add Comment: /post/:id/comment
  • Get User: /user/:id
