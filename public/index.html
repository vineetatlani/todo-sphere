Welcome to Todosphere

Todosphere is a social media app where you can create and manage todos, posts, and comments.

APIs

  • Login API

    Endpoint: /login

    Description: Authenticate user and generate access token

    Method: POST

    Request Body: email and password

    Response Body: access token

  • User API

    Endpoint: /users

    Description: Manage user accounts

    Method: GET, POST, PATCH, DELETE

    Request Body: user data (for POST and PATCH)

    Response Body: user data

  • Todo API

    Endpoint: /todos

    Description: Manage user's todos

    Method: GET, POST, PATCH, DELETE

    Request Body: todo data (for POST and PATCH)

    Response Body: todo data

  • Post API

    Endpoint: /posts

    Description: Manage user's posts

    Method: GET, POST, PATCH, DELETE

    Request Body: post data (for POST and PATCH)

    Response Body: post data

  • Comment API

    Endpoint: /comments

    Description: Manage comments on user's posts

    Method: GET, POST, PATCH, DELETE

    Request Body: comment data (for POST and PATCH)

    Response Body: comment data
